# Home Lab Repository

Цей репозиторій містить конфігурації, скрипти, документацію та діаграми для моєї домашньої лабораторії.

## Структура
- `docs/` — текстова документація та інструкції
- `configs/` — конфігураційні файли (Ansible, Terraform, Docker, сервіси)
- `scripts/` — Bash/Python скрипти для автоматизації
- `pipelines/` — файли CI/CD
- `diagrams/` — топології мереж та архітектурні схеми
- `tests/` — тести та результати експериментів
- `archive/` — старі конфіги та налаштування

## Використання
1. Клонуй репозиторій
2. Заповни інструкції в `docs/`
3. Додай свої конфіги та скрипти
4. Веди зміни через Git для історії
